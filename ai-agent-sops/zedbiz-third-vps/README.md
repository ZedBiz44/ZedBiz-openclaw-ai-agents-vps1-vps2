# ZedBiz Third VPS

This folder tracks work on the third VPS, including domain-specific agents and experiments that do not belong on the main Docker VPS or the secondary VPS.

## Track Here

- third VPS setup and changes
- domain-specific agent setup
- tools
- client or niche workflow guides
- skill additions
- server-level changes
- migration or test results

## Required Log Rule

Every meaningful change should create or update one tracking entry in `tracking/`.
