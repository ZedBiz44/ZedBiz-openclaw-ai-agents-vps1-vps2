# Agent SOP Library

This folder holds the live, tested SOPs for creating, configuring, fixing, and operating ZedBiz AI agents.

## Recommended SOP Groups

- `agent-creation/` - Creating new agents from the base SOP.
- `agent-configuration/` - Identity files, prompts, tools, skills, memory, and environment setup.
- `routing-and-domains/` - Caddy, domains, Cloudflare, ports, and access checks.
- `vps-operations/` - Docker, permissions, folder structure, backups, and health checks.
- `troubleshooting/` - Known failures, confirmed fixes, and diagnostic paths.

## Status Labels

- `draft` - Useful, but not fully tested.
- `tested` - Confirmed working in the real environment.
- `needs-fix` - Something failed and needs repair.
- `deprecated` - Kept for history, no longer recommended.
