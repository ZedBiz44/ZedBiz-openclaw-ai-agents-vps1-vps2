# Record Quality Gate

Use this checklist before reporting completion.

## Source Integrity

- Original imported page ID retained.
- Parent is the intended Content Master data source.
- Original body matches the pre-change snapshot or fingerprint when preservation was required.
- No imported blocks, attachments, child pages, or databases were silently removed.
- Any cleaned or synthesized derivative is separate from the preserved source.

## Classification And Identity

- Correct database, Page-Type, current template, and naming rules.
- Existing topic lookup completed before allocation.
- Full topic identity uses Core + Lane + Topic Identifier, not the six-digit number alone.
- Z-Code came from the allocator.
- Minimal page verification occurred before allocator confirmation.
- No duplicate complete Z-Code or conflicting full topic identity.

## Required Operating Fields

- Every live-SOP required field is complete.
- Optional values are supported by evidence.
- Exact live select values are used.
- Unknown values are empty or explicitly marked only when the schema supports that choice.

## Evidence And Relationships

- Source statements, verified facts, conclusions, and recommendations are distinguishable.
- Confidence, risks, unknowns, and verification gaps are honest and useful.
- People, Business, Tool, Decision, and other relations were identity-checked before linking.
- Related assets connect only verified records.
- Biz-Plan exists only when an executable planning artifact is justified.

## Presentation And Verification

- Topical icon renders after refetch.
- New authored content has the required date, agent, and status line using Mountain Time.
- Every changed Notion record was fetched after writing.
- Every Core Content Database record has a corresponding Memory Wiki record.
- The exact Wiki artifact exists and passed the applicable status, standard lint, and ZedBiz frontmatter checks.
- One compact activity-memory record was updated and its write verified.
- Completion report names records deliberately not created and why.
