# Decision Log

Use this file for decisions that affect how ZedBiz agents are created, documented, or operated.

## Current Decisions

### GitHub And Notion Split

- **Decision:** GitHub stores the live SOP content and detailed change history. Notion stores dashboard-level visibility and summaries.
- **Reason:** SOPs change like software. GitHub gives clear version history, issues, branches, and commit messages.
- **Status:** Active

### Rapid Execution Mode

- **Decision:** Build the simplest working version first, test in the real environment, then improve.
- **Reason:** This protects time and surfaces real problems faster than over-planning.
- **Status:** Active
